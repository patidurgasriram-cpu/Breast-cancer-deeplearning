🩺 Breast Cancer Classification using Feedforward Neural Network (FFNN)

This project implements a **Feedforward Neural Network (FFNN)** using **PyTorch** to classify breast cancer tumors as **Benign** or **Malignant** using the **Wisconsin Diagnostic Breast Cancer Dataset**.

📌 Features

- Feedforward Neural Network (FFNN)
- 5 Hidden Layers (10 Neurons each)
- ReLU Activation
- Sigmoid Output
- Batch Normalization
- Dropout Regularization
- Early Stopping
- Adam & SGD Optimizers
- Binary Cross Entropy & MSE Loss Comparison
- ROC Curve & AUC
- Confusion Matrix

📂 Dataset

Wisconsin Diagnostic Breast Cancer Dataset

https://archive.ics.uci.edu/dataset/17/breast+cancer+wisconsin+diagnostic
